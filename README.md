# NinixFMO

A gem for using FileMappingObject in ninix-kagari.

## Requirements

```
apt install build-essential rake rake-compiler ruby-dev
```

## How to Install

```
git clone https://github.com/Tatakinov/ninix-fmo.git
cd ninix-fmo
rake install
```
